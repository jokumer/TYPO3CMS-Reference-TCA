customControls
~~~~~~~~~~~~~~

:aspect:`Datatype`
    array

:aspect:`Scope`
    Display

:aspect:`Description`
   Numerical array containing the user function to call. The user function must return HTML.
   This makes it possible to create custom header controls for IRRE fields.
   
   The user function string is defined as usual in TYPO3 as ['class']->['method'], e.g. ::

      Vendor\\Extension\\MyClass::class . '->myUserFuncMethod'

   For more details, see the implementation in :php:`TYPO3\\CMS\\Backend\\Form\\Container\\InlineControlContainer`
   and search for "customControls".
